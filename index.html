What is Nodeschool?

NodeSchool is an open source project run by volunteerst with two goals: to create high quality programming curriculum and to host community learning events.

Workshoppers

Workshopper is the name used for the open source lesson modules associated with NodeSchool. All are self guided (you don't need to attend a workshop to do one) and most work offline.

Workshops

These are free to attend, in-person hosted workshops, in which workshoppers are used as curriculum and mentros help attendees work through the challenges. We host NodeSchool in Haarlem two times a month*

* obligatory pizza and drinks included, for Free.
