Before we get started…

Make sure you’re well prepared and have node, npm and git as well as a text editor installed on you computer

Installing node and npm

The easiest way to install node and npm… is to visit https://nodejs.org/en/download/… and download an installer for your system and run it

Installing git

To install git on your computer… go to
http://www.git-scm.com/downloads… and download the installer for your system and run it

Using a package manager

Feel free to install node, npm and git with a package manager like brew or apt-get, but make sure you read the instructions carefully!

Check if it all works

To check if everything works, open a console and type:

 $ node -v
 v6.11.0
 $ npm -v
 v3.10.10
 $ git --version
 git version 2.13.1

If you’re having problems, ask one of the mentors to take a look and help sort it out

Installing a text editor

Any text editor will do, but we recommend using Atom (it’s build using node!), which you can download from https://atom.io/

Got everything installed?

Awesome! Get yourself an account on GitHub and join the developer community!

This is optional, but will allow you to get more out of NodeSchool

Signing up for GitHub

Go to https://github.com/ and fill out your details…, verify your account by following the instructions in the email… and presto, you’re done!

You’re ready for NodeSchool to begin!
